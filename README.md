# 회원제 게시판 프로젝트



## ✔ 주요기능
___
***회원 관련 기능***
- 회원가입
- 로그인
- 로그아웃
- 일반회원: 게시글 작성, 조회, 수정, 삭제
- 관리자: 회원목록조회, 회원삭제

***게시판 관련 기능***
- 글쓰기
- 페이징 목록 출력
- 글 수정, 삭제
- 작성자, 제목 검색
- 댓글 작성

***마이페이지 관련 기능***
- 회원정보 수정
